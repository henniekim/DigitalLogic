# 디지털논리회로실험 
EEE2052를 support하기 위한 repository 입니다. 실험 교재는 별도로 포함하지 않으며, schematic 파일과 vhdl 코드 등을 업로드 합니다. 

# 실험 환경 
- Windows 10 (실행 오류시 패치 방법 [클릭]()) 
- ISE 14.7 
- Spartan3 XC3S2000
